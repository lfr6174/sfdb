# 編輯指引

## 收錄準則

本站的收錄對象暫定為戰後出版/獲獎的本土科幻作品，各詞彙的涵義詳述如下，

- 科幻：被當作科幻來行銷討論、讓人聯想起既有科幻作品、透過推演領域知識創造殊異感的著作。
- 本土：在臺發跡或與臺灣文壇關係密切的作者，不限其國籍與居住地點。
- 作品：小說、漫畫及詩文等體裁的想像作品，排除書評、訪談、論文、序跋、導讀等圍繞作品的非虛構論述。
- 出版：經第三方評選或出版，非個人/組織自行發布的著作。
- 戰後：對應 1968 年張曉風發表〈潘渡娜〉開啟臺灣科幻小說先聲。

以上準則的設計是為了以有限的維護人力，盡可能和同性質的網站做出區別。待本站運作穩定，便有機會擴大收錄範圍，納入翻譯代理或其它體裁的作品。
